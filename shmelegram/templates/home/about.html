What is Shmelegram?

Shmelegram is a messenger. It was insipred by famous Telegram messenger. It's name was created by mixing a mocking "sh-" prefix and Telegram name. And by a coincidence the name rhymed with "shmel" (bee specie in russian).

The website was created in Python with help of Flask web framework. In addition, for chat it uses SocketIO, for database - SQLAlchemy with MySQL under the hood. Client side uses pure JavaScript for chat to work.

The messenger was created by students Oleg Yevtushok and Artem Sukach, with the help of documentation, books, Youtube tutorials, courses and so much more...
